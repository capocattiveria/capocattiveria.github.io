Detailed Description

template<class T>
class slothLib::Graph< T >

A graph data structure template.

This class provides a basic implementation of a graph data structure. It supports adding and removing nodes and edges, as well as performing basic graph operations such as breadth-first search and depth-first search.

Author
Nico Fiorini

TThe type of the nodes in the graph.

Detailed Description

template<class T>
class slothLib::GraphEdgeList< T >

A data structure for a graph implemented with an edge list technique.

This class represents a graph using an edge list data structure. It inherits from the Graph class and provides methods to work with the graph.

Template Parameters
TThe type of data associated with the nodes in the graph.

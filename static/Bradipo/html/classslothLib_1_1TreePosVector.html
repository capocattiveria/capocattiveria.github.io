Detailed Description

template<class T>
class slothLib::TreePosVector< T >

A Tree data structure implemented with vector positions.

The TreePosVector class represents a tree data structure implemented using vector positions. It imposes structural constraints on the maximum node degree and the maximum tree height.

Template Parameters
TThe type of the nodes in the tree.

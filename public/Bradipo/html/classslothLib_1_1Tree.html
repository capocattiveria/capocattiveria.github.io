Detailed Description

template<class T>
class slothLib::Tree< T >

A generic tree data structure for representing hierarchical relationships.

This class is a generic tree data structure that represents hierarchical relationships between nodes. It extends the BasicGraph class and provides methods for adding and manipulating nodes within the tree.

Author
Nico Fiorini
